# Nav2-Path-Planner



## A* 算法路径规划

### 1 编译

```shell
colcon build --packages-up-to nav2_demos
```



### 2 运行

```shell
source install/setup.zsh 
or
source install/setup.bash
```



### 3 设置A点和B点，AB点路径规划

点击A点和B点，进行路径规划，可以选择自定义的地图
